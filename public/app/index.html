металлопроката в браузере

Возможности

Два режима — один инструмент

От сечения профиля до готового ЧПУ-файла. Всё считается локально в браузере, данные не покидают ваш компьютер.

Развёртка профилей

Расчёт плоской развёртки по нейтральной оси: K-фактор определяется автоматически по таблице R/S, длина дуги и прямые участки считаются с учётом setback.

  • Стандартные профили: уголок (L), швеллер (U), G и C
  • Конструктор произвольного сечения: сегменты и углы гиба
  • K-фактор по таблице R/S — как в справочниках
  • Визуализация сечения и линий сгиба

Оптимизация раскроя (nesting)

Алгоритм MaxRects-BSSF упаковывает детали на листе с учётом припуска на рез (kerf) и отступов от краёв. Никаких пересечений и выходов за пределы.

  • Несколько листов и неограниченный список деталей
  • Блокировка поворота для учёта направления волокна (grain)
  • Подсветка отходов и расчёт использования материала
  • Таблица координат (X, Y, угол) для ЧПУ

Экспорт в CAD

DXF с сущностями LWPOLYLINE и LINE открывается в AutoCAD и Компасе. SVG — для веба и вёрстки. PDF — для печати и документации.

Конструктор сечений

Визуальный редактор 2D-сечения: прямые отрезки, произвольные углы загиба, радиусы и вырезы.

Приватно и офлайн

Все расчёты — в браузере. Сессии и развёртки сохраняются в localStorage. Сервер не нужен.

Математика

Точный расчёт по нейтральной оси

Длина развёртки считается по нейтральной оси с учётом K-фактора, который определяется автоматически по отношению R/S — как в справочниках по листовой гибке. Это даёт корректную длину заготовки для пресса.

Bend Allowance
Припуск на гиб
Li = (π / 180) · A · (R + K · S)

Li — длина дуги нейтрального слоя при гибке.
A — угол гиба в градусах, R — внутренний радиус, S — толщина металла.
K — коэффициент положения нейтральной оси, определяется по таблице R/S (0.33 при R/S→0, 0.5 при R/S≥5).

Полная длина
Развёртка
L = L1 + L2 + … + Ln + n · Li
Lкрай = A − (R+S)·tan(Aгиб/2)

L — полная длина развёртки = сумма прямых участков + n дуг.
Крайние полки (1 гиб) вычитают setback один раз, промежуточные (2 гиба) — дважды.

Как это работает

Четыре шага до ЧПУ-файла

01 / ВВОД
Задаёте профиль

Выбираете стандартный профиль или собираете произвольное сечение. Указываете толщину, радиус и угол гиба.

02 / РАСЧЁТ
Считаете развёртку

K-фактор по таблице R/S применяется автоматически. Видите сечение, прямые участки и линии сгиба.

03 / РАСКРОЙ
Укладываете на лист

Переносите развёртки в раскрой, задаёте kerf и отступы. MaxRects упаковывает детали на листах.

04 / ЭКСПОРТ
Сохраняете в CAD

DXF для ЧПУ, SVG для веба, PDF для печати. Координаты — в таблице для станка.
