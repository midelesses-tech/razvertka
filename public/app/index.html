Возможности

Три режима — один инструмент

От сечения профиля до готового ЧПУ-файла. Всё считается локально в браузере, данные не покидают ваш компьютер.

Развёртка профилей

Расчёт плоской развёртки по нейтральной оси: K-фактор определяется автоматически по таблице R/S, длина дуги и прямые участки считаются с учётом setback.

  • Стандартные профили: уголок (L), швеллер (U), G, C, хомут
  • Конструктор произвольного сечения: сегменты и углы гиба
  • K-фактор по таблице R/S — как в справочниках
  • Визуализация сечения и линий сгиба

Оптимизация раскроя (nesting)

Алгоритм MaxRects-BSSF упаковывает детали на листе с учётом припуска на рез (kerf) и отступов от краёв. Никаких пересечений и выходов за пределы.

  • Несколько листов и неограниченный список деталей
  • Блокировка поворота для учёта направления волокна (grain)
  • Подсветка отходов и расчёт использования материала
  • Таблица координат (X, Y, угол) для ЧПУ

Металлокалькулятор

Вес и длина по 13 видам проката: круг, труба, квадрат, уголок, швеллер, двутавр, лист и др. 7 материалов с плотностью.

Экспорт в CAD

DXF с сущностями LWPOLYLINE и LINE открывается в AutoCAD и Компасе. SVG — для веба. PDF — для печати.

Приватно и офлайн

Все расчёты — в браузере. Сессии и развёртки сохраняются в localStorage. Сервер нужен только для аккаунта.

Математика

Точный расчёт по нейтральной оси

Длина развёртки считается по нейтральной оси с учётом K-фактора, который определяется автоматически по отношению R/S — как в справочниках по листовой гибке. Это даёт корректную длину заготовки для пресса.

Bend Allowance
Припуск на гиб
Li = (π / 180) · A · (R + K · S)

Li — длина дуги нейтрального слоя при гибке.
A — угол гиба в градусах, R — внутренний радиус, S — толщина металла.
K — коэффициент положения нейтральной оси, определяется по таблице R/S (0.33 при R/S→0, 0.5 при R/S≥5).

Полная длина
Развёртка
L = L1 + L2 + … + Ln + n · Li
Lкрай = A − (R+S)·tan(Aгиб/2)

L — полная длина развёртки = сумма прямых участков + n дуг.
Крайние полки (1 гиб) вычитают setback один раз, промежуточные (2 гиба) — дважды.

Как это работает

Четыре шага до ЧПУ-файла

01 / ВВОД
Задаёте профиль

Выбираете стандартный профиль или собираете произвольное сечение. Указываете толщину, радиус и угол гиба.

02 / РАСЧЁТ
Считаете развёртку

K-фактор по таблице R/S применяется автоматически. Видите сечение, прямые участки и линии сгиба.

03 / РАСКРОЙ
Укладываете на лист

Переносите развёртки в раскрой, задаёте kerf и отступы. MaxRects упаковывает детали на листах.

04 / ЭКСПОРТ
Сохраняете в CAD

DXF для ЧПУ, SVG для веба, PDF для печати. Координаты — в таблице для станка.

Вопросы и ответы

Часто задаваемые вопросы

Нужна ли регистрация для использования? +

Нет. Базовые функции (расчёт развёртки, раскрой, SVG/DXF-экспорт) доступны без регистрации. Личный кабинет нужен только для сохранения проектов между устройствами и оформления премиум-тарифа.

По какой формуле считается развёртка? +

Развёртка считается по нейтральной оси:

L = Σ прямых участков + n · Li

где Li = (π/180)·A·(R + K·S) — длина дуги одного гиба, K — коэффициент положения нейтральной оси, определяемый по таблице R/S (DIN 6935): 0.33 при R/S→0, 0.5 при R/S≥5.

Можно ли экспортировать результат в AutoCAD или Компас? +

Да. Экспорт в DXF (AutoCAD R12/ASCII) использует сущности LWPOLYLINE и LINE на отдельных слоях (BEND, SHEET, PARTS). Файл открывается в AutoCAD, Компасе и других CAD. Дополнительно доступен SVG для веба и PDF для печати.

Работает ли инструмент офлайн? +

Все расчёты выполняются в браузере — данные не покидают компьютер. Сервер нужен только для личного кабинета и сохранения проектов между устройствами. После первой загрузки приложение работает и без интернета.

Какие тарифы доступны? +

Демо на 24 часа (100 ₽), PRO на месяц (1000 ₽) и TEAM на год (2999 ₽). Демо-режим можно активировать мгновенно — это удобно для разовой работы. PRO и TEAM открывают PDF-экспорт, несколько листов в раскрое, безлимит проектов и конструктор сечений.

Готовы рассчитать первую развёртку?

Никакой регистрации и установки. Откройте инструмент и начните — всё считается прямо в браузере.
